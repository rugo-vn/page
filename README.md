# Rugo Page

## Install

```bash
npm i -D @rugo-vn/page
```

Add scripts:

```json
{
  "scripts": {
    "dev": "rugopa",
    "build": "rugopa --build"
  }
}
```

Run:

```bash
npm run dev
```

Build:

```bash
npm run build
```

## 3rd libraries

**TailwindCSS**

Install TailwindCSS using PostCSS.
